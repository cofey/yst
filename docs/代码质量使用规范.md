# 代码质量使用规范

适用范围：`yst-server/`、`yst-ui/`。

## 首次启用 Hooks

首次拉取仓库后执行一次：

```bash
git config core.hooksPath .githooks
```

校验是否生效（应输出 `.githooks`）：

```bash
git config --get core.hooksPath
```

## 本地执行命令

后端全量检查：

```bash
cd yst-server && mvn clean verify
```

前端检查：

```bash
cd yst-ui && pnpm run check && pnpm run type-check
```

前端自动修复：

```bash
cd yst-ui && pnpm run lint:fix && pnpm run format:write
```

与 `git push` 一致的门禁检查：

```bash
./.githooks/pre-push
```

## 推荐顺序

```bash
cd yst-ui && pnpm run lint:fix && pnpm run format:write
cd yst-server && mvn clean verify
cd yst-ui && pnpm run check && pnpm run type-check
```
