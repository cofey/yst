# 前端分页贴底布局约定

适用目录：`yst-ui/src/modules/**/pages/*.vue` 中的列表页。

## 目标

列表页在数据较少时，分页区域也应保持贴在主卡片底部，避免分页悬空在中间。

## 结构约定

列表页建议遵循以下结构（简化示意）：

```vue
<div class="page">
  <el-card>
    <!-- 查询区/工具栏/表格 -->

    <div class="pager-wrap">
      <el-pagination ... />
    </div>
  </el-card>
</div>
```

满足该结构时，分页贴底由全局样式自动生效，无需页面单独写定位样式。

## 全局样式约束

全局样式定义在：`yst-ui/src/styles/global.css`

- `.page` 使用纵向 flex 容器。
- `.page > .el-card` 与 `.el-card__body` 纵向撑满。
- `.pager-wrap` 使用 `margin-top: auto` 将分页推到底部。

## 注意事项

- 若页面存在多个 `el-card`，仅主列表卡片应使用上述结构。
- 若页面需要自定义分页位置，请避免复用 `.pager-wrap`，改用独立 class。
